- 作業中にpythonコードを実行したい場合は、`python -c` を使うのではなく、一時的なスクリプトファイルを作成して実行してください。
  - ファイルは一時フォルダ `$(BASEDIR)/.AIdir/.tmp` に記録してください。`rm` は行わなくて大丈夫です。
- スクリプトを作成する際には、1つの機能だけに特化したものとして、複数の機能が必要な場合は複数のスクリプトを作成するようにしてください。
- conda 環境については、既にあるものではなく、必要なものを入れた tmp 環境を作成してください。
  - この時、常に一時的な `environment.yml` を作成し、その中身は log にも記述するようにしてください。
  - `conda create -y -n tmp[YYYYMMDDHH] -c conda-forge -f environment.yml` を実行することで、インストールを完了させてください。
  - 環境のコンタミネーションを回避するために、`pip install` は極力使わないようにしてください。どうしても使わなければならない場合は問い合わせてください。
- conda環境は activate せずに直接呼び出してください。
  - TSUBAME4 (hostname `r*n*` )：`~/.local/anaconda3/envs/[ENVNAME]/bin/python` 
  - 研究室サーバ (hostname `harp01` )： パスを問い合わせてください。
  - その他のサーバについてはパスを問い合わせてください。
